# 🌀 COCO - AI 학습메이트

수학 개념과 실생활을 연결하는 스토리 기반 탐구 학습 앱

## 소개
중학교 수학(연립방정식·외심·일차함수·확률)을 학교 현장의
실제 상황과 연결하여 탐구하는 React 기반 PWA입니다.

## 시작하기

\`\`\`bash
npm install
cp .env.example .env   # 그리고 .env에 본인의 API 키 입력
npm start
\`\`\`

## 필요한 API 키
- Firebase 프로젝트 (firebase.google.com)
- Gemini API 키 (aistudio.google.com)
